Vagrantの基本的な使い方

Vagrantはハイパーバイザー上に仮想マシンを構築するツール。

インストール

以下サイトより任意のバージョンのパッケージをダウンロードし、インストール。
vagrantup.com

コマンド

初期化

% vagrant init

vagrant init <BOX名> <BOXのURL>でBOXを指定して初期化することも出来る。
BOXとは仮想マシンのテンプレートのような物。

BOXの制御

  • BOX追加
% vagrant box add <BOX名> <BOXのURL>

BOXの一覧に新たなBOXを追加する。
BOX名は任意の名前を付けられる。

以下サイトよりBOXのURLを取得可能。
Vagrantbox.es

  • BOXの一覧表示
% vagrant box list

仮想マシン起動

% vagrant up

仮想マシンの一時停止

% vagrant suspend

再開する場合はvagrant up。

仮想マシンの停止

% vagrant halt

ホストマシンを終了させる場合、必ず仮想マシンを停止させる。

仮想マシンの再起動

% vagrant reload

仮想マシンの破棄

% vagrant destroy

仮想マシンの状態確認

% vagrant status

仮想マシンへのSSH接続

% vagrant ssh

仮想マシンのSSH情報表示

% vagrant ssh-config

vagrant ssh-config >> ~/.ssh/configと入力することで、sshコマンドでの接続が可能に。
